Переменние

Переменние використовуються для збереження інформації

Переменние створюються за доп var(старий метод), або let(новий спосіб)

Переменні можуть бути як let,var або cons,const-це переменна яку не можна в подальшому змінювати

В переменній може зберігатись, як речення, так і число або булеан

let num = 9;
let string = 'Hello world'

Імена переменних: можуть бути цифри і букви, но не можна використовувати цифри в початку назви

Також не можна використовувати зарезервовані імена такі як: var, let,return,export

Взаємодія з користувачем

alert("Hello world")-виводить модальне повідомлення на екран, зупиняючи скріпт.

prompt("Скільки тобі років" )- виводить вікно в якому користувач може написати текст

confirm("Text")- виводить вікно, яке повертає значення true або false

Масив

Масив це різновидність обєкта, який предназначен для зберігання пронумерованих значень.Зазвичай використовується для зберігання упорядочних колекцій данних(список товарів, товарів для покупки); В ньому може зберігатись як число і строки, так і обєкти, і булеві значення

Спосіб створення масива:
let masuv = ['Хліб','Чай', 'Цукор'];

Щоб получити елемент з масива використовують alert(masuv[1])

Щоб замінити елемент масива: masuv[1]= 'Кава'; длбавляється елемент так само, тільки в душках записується номер, який буде створений

alert(masuv.length)-виводить число елементів
alert(masuv)-виводить цілий масив

masuv.pop();-удаляє елемент з кінця масива

masuv.push('Цукор')- добавляє в кінець масива елемент

masuv.shift -удаляє елемент з початку масива

masuv.unshift добавляє елемент з початку масива

Методи масивів

split()

let names = "Катя, Коля, Ваня, Міша"
names.split(", ") - split - перетворює рядок в масив, в душках це тек як він буде розділяти його

join()

let names = ["Катя", 'Коля', 'Ваня', "Міша"]
names.join(',')- join - перетворює масив в рядок

delete

let names = ["Катя", 'Коля', 'Ваня', "Міша"]
delete names[1] - delete - удаляє елемент в масиві, но місце елемента залишається

splice()

Удаляє, добавляє, заміняє елементи масива

names.splice(1, 1) -це означає, що начинаючи з позиції(від 0 до %) один удалити перший елемент

names.splice(2, 0, 'Я', 'Иду')- це означає, що починаючи, з другої позиції, не удаляючи нічого, но добавити дві строки

reverse()

names.reverse();- обертає масив задом на перед, якщо він складається із цифр

concat

let newNames = names.concat('Назва ',' Какашки')- обєднює масиви
